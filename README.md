# DIGITAL-LITERACY-PROJECTS

Overview

This repository contains all modules and tasks for the Digital Literacy project. As a Student Digital Ambassador, I completed activities to enhance digital skills, create awareness about safe online practices, and build professional online presence. The tasks include infographics, digital portfolios, coding practice, professional email drafting, and cybercrime awareness.

Task 1 – Digital Literacy Awareness Infographic

Module: 1 | Marks: 20
Description: Created a one-page infographic using Canva covering:

What digital literacy is
Useful digital tools for students
Safe internet practices

Reflection (150–200 words): The infographic visually explains digital literacy concepts for peers. I found designing clear visuals and balancing information challenging, but it helped me understand how to communicate concepts effectively.
